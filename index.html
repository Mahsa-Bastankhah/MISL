mb6458@princeton.edu· sophiebroderick@princeton.edu· eysenbach@princeton.edu

Abstract

  • Learning representations that only capture the control-relevant information is crucial in high-dimensional environments.
  • Current methods either learn representations from an expert dataset (hard to collect) or use task-specific reward functions (not generalizeable).
  • In this work, we theoretically show Empowerment, an objective historically used for diverse policy training in online settings is a principled way to learn representations that capture only the control-relevant information.
  • Intuitively an empowerment agent only cares about controlling the environment therefore the representations it learns are only focused on control.
  • We show that empowerment agents learn two set of representations, forward and backward. The forward representation captures what states are reachable from a state and the backward captures how a state is reached. Both of these representations are invariant to noise.
  • We verify our theoretical claims by empirical results in high dimensional and pixel-based environments.

Presentation

Empirical Results

Empowerment representations don't change by changing the noise component of the state (invariance) but they significantly vary by changing the control-relevant features.
Using empowerment representations for training policies in noisy environments significantly improves the performance.

Geometric Intuition

In the simplex of future-state distributions over (s0, s1, s2, s3), each action is represented by a point. If we change the control-relevant part of the initial state, x0, the simplex changes completely (Top). This means that two initial states with different control-relevant components induce fundamentally different future-state distributions. In contrast, if we only change the control-irrelevant part of the initial state (e0), the resulting future-state distributions are simply a rotated and shifted version of the each other (Bottom). Since the geometry remains the same, empowerment picks the same actions in the rotated and shifted simplex, actions that maximize the divergence between the the average future state distribution and each individual action future state distribution.
